# Roadmap

## Features

### User Auth
OIDC authentication to pass user info. 

### Linking web page to Jira
Attach links to Jira tickets tracking software approval. 
App can generate urls of info of whether is passed or not. 

### Scan info
read generated.json scan info and place it into db
Scan info will be a dropdown option from file UI that can be exportable. 

### API Keys
User API keys to allow anyone to use api to upload.

## Permissions

### What users can see? 
Users should be able to see only files that they uploaded. Anyone can upload a file after they authenticate.

### Admin
Can view everything within and can manage user groups.

### Auditors
Can view all files update status on file
